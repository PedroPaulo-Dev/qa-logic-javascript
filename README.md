# 🚀 JavaScript para QA: Da Lógica ao Cypress (BDD & Deep Dive)

Este repositório é o meu "laboratório de testes". Aqui, o foco não é apenas automatizar, mas entender a **engrenagem** por trás dos sistemas. Utilizo JavaScript — ou para os íntimos, o **ECMAScript** — para elevar o nível da garantia de qualidade.

## 🎯 O que este projeto resolve?
Muitas vezes, o erro é intermitente porque não entendemos a lógica do sistema. Este projeto foca em:
* **Deep Logic:** Funções JS para entender como os dados fluem antes mesmo de abrir o navegador.
* **Cypress Selectors:** Práticas de mapeamento de elementos de forma performática e resiliente.
* **BDD com Gherkin:** Escrita de casos de teste que o negócio entende e a automação executa.
* **Mão na Massa:** Código real para investigação de falhas e manipulação de massa de dados.
